## Инструкция

1. Клонируйте репозиторий 
    ```bash
    git clone https://github.com/ravesjs/countries-app
    ```
     
    ## Установка
    
    Установить зависимости
    ```bash
    npm install
    ```

    ### Запуск проекта
    ```bash
    npm start
    ```
    
    ### Production
    
    ```bash
    npm run build
    ```
